Insertion Sort
Core Idea:

Assuming we're looping from the array's start:
For each element, move it to the left until [left <= current].

Sample Implementation:

    int* a = [3,6,2];
    int tempNum;
    int curPos = 0;
    for(int i = 0; i < a.length;i++){
        curPos = i;
        while(curPos > 0 || a[i] < a[curPos - 1]){
            curPos--;
        }
        if(curPos != i){
            //swaps a[i] and a[curPos]
        }
    }
Extras & Gotchas
  • Costly if the element sits too far from its sorted position.
